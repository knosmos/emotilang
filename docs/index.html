an intuitive scripting language

Emotilang (short for "emoticon language") is a versatile scripting language where every character is emoji. Because of this, there are no language barriers. Printing text to the command line is just 🖨️, and the curly braces of most programming languages are replaced with a far more intuitive 🏁 and 🛑. By having every command have a physical connection, Emotilang is extremely intuitive and easy to learn.

documentation

DATA TYPES

Emotilang can handle two types of data: string and integer. Strings are always lowercase. Each character is represented as an emoji:

abcdefghi jkkm
🅰️🅱️©️↩️📧🎏⛽️♓️🕯️ 🗾🎋👢♏️
nopqr stuvwxyz
♑️🅾️🅿️🧶®️ 💲✝️⛎♈️〰️❎💹⚡️
space?!.,
✨❓❗🧿🧨

Strings are always enclosed with 📜 characters. For example, “hello world” is represented as the string 📜♓️📧👢👢🅾️✨〰️🅾️®️👢↩️📜.

Integers are written out as digits enclosed with 💯 characters, where each digit is represented as a clock symbol:

0123456789
🕛🕐🕑🕒🕓🕔🕕🕖🕗🕘

The same characters are used to represent numbers inside a string.

To convert from a string to an integer, use the 🧮 function:

🧮📜🕐🕑📜 returns 💯🕐🕑💯

To convert from an integer to a string, use the ✉️ function:

✉️💯🕐🕑💯 returns 📜🕐🕑📜

VARIABLES

The valid variable names are 📦,🏺,⚗️,🧪,🗃️, and 🛍️ (these are all emoji of containers). Assignment is done with the ✏️ operator. The following code creates a variable with a value of 1337:

📦✏️💯🕐🕒🕒🕖💯

ARITHMETIC

Emotilang supports addition (➕), subtraction (➖), multiplication (✖️) and division (➗). Only one operation can be done per line, because I’m too lazy to write a proper parser. Example:

💯🕐🕛💯➕💯🕔💯 returns 15
💯🕐🕛💯➖💯🕔💯 returns 5
💯🕐🕛💯✖️💯🕔💯 returns 50
💯🕐🕛💯➗💯🕔💯 returns 2

COMPARISON

Two values can be compared with the 🧲 (greater than), ⚖️ (equals), and ⛔ (not) operators. A comparison statement cannot contain any arithmetic operations, and only one comparison statement can be done per line for now. Examples:

💯🕐🕛💯🧲💯🕔💯 returns True
💯🕐🕛💯⚖️💯🕔💯 returns False

📦✏️💯🕐🕛💯🧲💯🕔💯
⛔📦 returns False

INPUT

Input is done via the 💬 command. For example, this code queries “how are you” and stores the result in variable ⚗️:

⚗️✏️💬📜♓️🅾️〰️✨🅰️®️📧✨💹🅾️⛎📜

Note that the 💬 function returns a string.

OUTPUT

Printing to the terminal is done via the 🖨️ command. Example:

🖨️📜♓️📧👢👢🅾️✨〰️🅾️®️👢↩️📜

CONTROL FLOW

Emotilang has forever loops (♾️), if commands (🤔), and break commands (✋).

IF commands (🤔) cause code inside them (starting with 🏁 and ending with 🛑) to be executed if the statement after the 🤔 mark evaluates to true. For example, the following code prints “hi” if 5+4 = 9:

📦✏️💯🕔💯➕💯🕓💯
🤔📦⚖️💯🕘💯
🏁
    🖨️📜♓️🕯️📜
🛑

FOREVER loops (♾️) cause the content inside of them to run until program execution is manually stopped, or until a ✋ command is sent. The following code asks “how are you” repeatedly until the user answers “good”:

♾️
🏁
    📦✏️💬📜♓️🅾️〰✨🅰️®️📧✨💹🅾️⛎📜
    🤔📦⚖️📜⛽️🅾️🅾️↩️📜
    🏁
        ✋
    🛑
🛑
🖨️📜♑️🕯️©️📧✨✝️🅾️✨♓️📧🅰️®️📜

installation

Download Emotilang from the Github Repository. To run, enter ./emotilang.exe <filename> into the command line, where <filename> is the name of the emotilang file.
